* 常用插件
** Surfingkeys
   绑定一些常用浏览器功能到快捷键，类似 Vim 键绑定：
   | 功能                 | 快捷键 |
   |----------------------+--------|
   | 跳转到书签           | b      |
   | 将当前网页添加到书签 | ab     |
   | 打开 Google 搜索     | og     |
   | 切换到左边 Tab       | E      |
   | 切换到右边 Tab       | R      |
   | 当前页面后退         | S      |
   | 当前页面前进         | D      |
   | 关闭当前 Tab         | x      |
   | 打开最近关闭的 Tab   | X      |
   | 打开链接             | f      |
   | 转到输入框           | i      |
   | 搜索                 | /      |
